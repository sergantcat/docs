## Code Examples

Here are a couple of real examples that reflect where I am right now. Nothing overly complex — but real, working code that I've written and understand.

**Example 1 — A simple greeting function:**

```javascript
// Greeting function
function greet(name) {
  return `Hello, ${name}! Welcome to my portfolio.`;
}

console.log(greet("Visitor")); // Hello, Visitor! Welcome to my portfolio.
```

This was one of the first things I built when learning about template literals and function return values. It's simple, but understanding how string interpolation works with backticks was a genuine "aha" moment for me — especially coming from Lua's string concatenation with `..`.

**Example 2 — DOM manipulation on a button click:**

```javascript
// Change button text on click
const button = document.querySelector("#myButton");

button.addEventListener("click", function () {
  button.textContent = "Clicked!";
  button.style.backgroundColor = "#ef7025";
});
```

This example shows how I can select a real element from an HTML page and make it respond to user interaction. Updating the text and background colour on click was one of the first times I felt like I was building something genuinely interactive on the web, which was a really motivating moment.

## What I'm Learning Next

I've got a clear roadmap of where I want to take my JavaScript skills. Here's what's on my learning queue:

<AccordionGroup>
  <Accordion title="Async/Await & Promises">
    Most interesting real-world JavaScript involves waiting for something — a network request, a timer, a file load. I'm learning how Promises represent future values and how `async`/`await` syntax makes asynchronous code readable and manageable, rather than falling into callback chaos.
  </Accordion>
  <Accordion title="Fetch API & REST">
    Once I'm comfortable with async patterns, I want to start consuming real APIs using the Fetch API. Being able to pull data from a REST endpoint — a weather API, a game database, anything — and display it on a page feels like a huge milestone that will unlock so many project ideas.
  </Accordion>
  <Accordion title="Introduction to React">
    React is everywhere in modern web development, and I want to understand why. I'm looking forward to learning about components, props, and state — and how React makes building complex, dynamic UIs much more organised than manipulating the DOM by hand.
  </Accordion>
  <Accordion title="Node.js Basics">
    Taking JavaScript to the server side with Node.js is something I'm genuinely excited about. Running JavaScript outside the browser, handling files, and eventually building simple backend APIs would bring my web development skills full circle and open the door to more ambitious projects.
  </Accordion>
</AccordionGroup>
